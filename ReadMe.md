# Projeto de Sistema de Suporte com serviços de IA

Este projeto visa desenvolver um sistema integrado de suporte técnico para uma empresa de médio porte. A inteligência artificial (IA) será usada para sugerir soluções de forma automática ou encaminhar os chamados ao técnico adequado, com base nas intruções fornecidas pelo time de desenvolvimento.
